Reconnaissance Tactics

  • Configuration Files: Search for filetype:env, filetype:config, or filetype:yml with keywords like "password" or "api_key".
  • Directory Listings: Use intitle:"index of" to find exposed directories containing backups, logs, or sensitive files.
  • Login Portals: Combine inurl:admin or intitle:login with domain targeting to map authentication surfaces.
  • Sensitive Documents: Search for PDFs or DOCs with keywords like "confidential", "internal only", or "do not distribute".
  • Version Disclosure: Find software versions with queries like "powered by" "version" to identify outdated systems.

Google caches pages, so you may find content that has since been secured. Always verify current state.

Security-Focused Examples

Click the copy button to grab any query, then test it in Google (on authorized targets only).

  1. Exposed .env files with database credentials
    filetype:env "DB_PASSWORD"
  2. Configuration files containing API keys
    filetype:config "api_key" OR "apikey"
  3. Exposed directory listings for admin panels
    intitle:"index of" /admin/
  4. Login portals and admin interfaces
    intitle:"admin login" OR intext:"administrator login"
  5. Confidential PDF documents
    filetype:pdf "confidential" OR "internal use only"
  6. SQL database dumps
    filetype:sql "INSERT INTO" "VALUES"
  7. Log files with potential credentials
    filetype:log "password" OR "username"
  8. Exposed backup files
    intitle:"index of" (backup OR .bak OR .zip)
  9. phpMyAdmin installations
    intitle:"phpMyAdmin" "running on" inurl:phpmyadmin
  10. WordPress config files with credentials
    filetype:txt inurl:wp-config.php "DB_PASSWORD"
  11. SSH private keys accidentally exposed
    intitle:"index of" .ssh "id_rsa" OR "id_dsa"
  12. Webcams and IoT devices with default credentials
    inurl:"/view/index.shtml" OR intitle:"Live View / - AXIS"
  13. Exposed Git repositories
    intitle:"index of" ".git" "parent directory"
  14. Server status pages revealing information
    intitle:"Apache Status" "Apache Server Status for"
  15. Error messages revealing paths and versions
    intext:"sql syntax near" OR intext:"syntax error has occurred"
