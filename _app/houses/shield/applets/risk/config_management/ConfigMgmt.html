Configuration settings are the parameters that you can change to affect the security or the functionality of hardware, software, or firmware components on servers, workstations, scanners, printers, firewalls, routers, wireless access points, operating systems, and applications. The security parameters set up (registry settings; account, file, or directory settings; and settings for ports, protocols, and remote connections) create a secure configuration, or system lockdown. Any unapproved changes get blocked.
