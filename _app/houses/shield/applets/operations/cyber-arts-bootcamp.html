🎭 What is Hacking?

▼

Black Hat

Malicious hackers who exploit systems for personal gain, data theft, or disruption. Their actions are illegal.

White Hat

Ethical hackers who use their skills to improve security. They work with permission to find and fix vulnerabilities.

Grey Hat

Operate in the middle - may find vulnerabilities without permission but report them rather than exploit them.

Key Insight: The difference isn't skill - it's intent and authorization.

🛡 The CIA Triad

▼

The three pillars of information security that every system must protect:

Confidentiality

Only authorized users can access data. Achieved through encryption, access controls, and authentication.

Integrity

Data remains accurate and unaltered. Ensured via hashing, checksums, and digital signatures.

Availability

Systems and data are accessible when needed. Protected by redundancy, backups, and DDoS mitigation.

🌐 What is a Network?

▼

A network is a collection of devices connected to share data. Built with hardware and software.

LAN (Local Area Network)

Small geographic area - office, home, building. High speed, low latency.

WAN (Wide Area Network)

Large geographic area - cities, countries. The Internet is the largest WAN.

Components

Router: Routes traffic between networks
Switch: Connects devices in a LAN
Firewall: Filters traffic

📍 IP Addressing

▼
Concept Description Example
IP Address Unique identifier for a device on a network 192.168.1.100
Subnet Mask Divides IP into network and host portions 255.255.255.0
Default Gateway Router that connects LAN to other networks 192.168.1.1
DNS Translates domain names to IP addresses 8.8.8.8

🔐 Types of Encryption

▼

Symmetric Encryption

Same key for encryption and decryption. Fast but key distribution is challenging.

Example: AES-256

Asymmetric Encryption

Public key encrypts, private key decrypts. Slower but solves key distribution.

Example: RSA

# Hashing

▼

Hashing converts data into a fixed-size string. It's one-way - you cannot reverse a hash to get the original data.

Algorithm Output Size Status
MD5 128 bits Deprecated (collisions found)
SHA-1 160 bits Legacy only
SHA-256 256 bits Recommended

🔒 Linux File Permissions

▼

Permissions control who can read (r), write (w), and execute (x) files:

- r w x r - x r - -

Owner: rwx (7) | Group: r-x (5) | Others: r-- (4) = chmod 754

Terminal - Permissions
chmod 600 secret.txt
# Owner read+write only

chmod u+x script.sh
# Add execute for owner

chown analyst:security file.txt
# Change owner and group

🗝 OpenSSL Encryption

▼
Terminal - Symmetric Encryption
echo "SECRET DATA" > secret.txt
# Create a file to encrypt

openssl enc -aes-256-cbc -salt -in secret.txt -out secret.enc
# Encrypt with AES-256

openssl enc -aes-256-cbc -d -in secret.enc -out decrypted.txt
# Decrypt the file
Terminal - Hashing

🔍 Nmap - Network Mapper

▼

The essential tool for network discovery and security auditing:

Terminal - Nmap Scanning
nmap -sn 192.168.1.0/24
# Ping sweep - find live hosts

nmap -sV 192.168.1.100
# Service/version detection

nmap -O 192.168.1.100
# OS detection

nmap -A 192.168.1.100
# Aggressive scan (all of the above)

nmap -p 445 192.168.1.0/24
# Check for open SMB ports (WannaCry)

🦈 Wireshark - Packet Analysis

▼

Filter: Display specific traffic

Common Filters

dns - DNS queries
tcp.port == 445 - SMB traffic

🔑 Password Cracking

▼

Passwords are stored as hashes, not plaintext. Crackers try to find a password that produces the same hash.

Attack Type Method Speed
Dictionary Try common passwords from a wordlist Fast
Brute Force Try every possible combination Slow
Hybrid Dictionary + rules (password1, P@ssw0rd) Medium

🔪 John the Ripper

▼

Classic password cracking tool. Reads hashes and compares against wordlists.

Terminal - John the Ripper
openssl passwd -6 'password123'
# Generate a SHA-512 hash to crack

echo '$6$rounds=...' > hashes.txt
# Save hash to file

john --wordlist=/usr/share/wordlists/rockyou.txt hashes.txt
# Crack using rockyou wordlist

john --show hashes.txt
?:password123
# Display cracked passwords

Ethical Warning: Only crack passwords you have authorization to test. Unauthorized access is illegal.
