Copper vs Fiber: The Big Picture

Two fundamentally different approaches

üî∂

Copper (Electrical)

  • Transmits electrical signals
  • Susceptible to EMI/RFI interference
  • Limited distance (typically 100m max)
  • Less expensive, easier to terminate
  • Can carry PoE power
Best for: Horizontal runs, workstations, access layer
üíé

Fiber (Light)

  • Transmits light pulses
  • Immune to EMI/RFI interference
  • Much longer distances (km+)
  • Higher speeds possible
  • Cannot carry electrical power
Best for: Backbone, datacenter, long-haul, high-speed

Twisted Pair Categories

The workhorses of modern networking

üîÄ Why "Twisted"?

The wires are twisted together to cancel out electromagnetic interference (EMI). More twists per inch = better noise cancellation = higher speeds!

Category Max Speed Max Distance Frequency Common Use
Cat5 100 Mbps Legacy 100m 100 MHz Obsolete - don't use!
Cat5e 1 Gbps Standard 100m 100 MHz Home/Small office
Cat6 10 Gbps* Fast 100m (55m@10G) 250 MHz Office standard
Cat6a 10 Gbps Fast 100m 500 MHz Enterprise, PoE++
üìù Exam Tip

Cat6 can do 10 Gbps but only up to 55 meters! For full 100m at 10G, you need Cat6a.

Fiber Optic Cable Types

Single-Mode vs Multi-Mode

Single-Mode Fiber (SMF)

  • Core size: 8-10 microns
  • Single light path (laser)
  • Distance: Up to 100+ km
  • Higher cost, precision alignment
  • Yellow jacket (typically)
Best for: Long-haul, WAN, campus backbone

Multi-Mode Fiber (MMF)

  • Core size: 50 or 62.5 microns
  • Multiple light paths (LED/VCSEL)
  • Distance: Up to 2 km
  • Lower cost, easier to terminate
  • Orange/Aqua jacket (typically)
Best for: Datacenter, building backbone, shorter runs
üî¶ Flashlight Analogy

SMF = Laser pointer - single focused beam, travels far without spreading. MMF = Flashlight - multiple beams bounce around, spreads out over distance (modal dispersion).

Power over Ethernet (PoE)

One cable for power AND data

üí° Why PoE?

Install IP phones, cameras, and access points without running separate power cables. Just one Ethernet cable does it all!

Standard IEEE Max Power (per port) Typical Devices
PoE Type 1 802.3af 15.4W (12.95W to device) IP phones, basic cameras
PoE+ Type 2 802.3at 30W (25.5W to device) PTZ cameras, access points
PoE++ Type 3 802.3bt 60W (51W to device) Video phones, high-power APs
PoE++ Type 4 802.3bt 100W (71W to device) Laptops, thin clients, displays

PoE Benefits

  • Simplified installation
  • Centralized power management
  • UPS backup for all devices
  • Remote power cycling

PoE Considerations

  • Switch power budget limits
  • Cable quality matters (Cat5e minimum)
  • Heat generation in cable bundles
  • Maximum distance still 100m

Plenum vs PVC vs Riser

Fire safety ratings matter!

CMP / Plenum

Plenum-Rated

  • Fire-resistant jacket
  • Low smoke emission
  • No toxic fumes when burned
  • Required in: Air handling spaces (above drop ceilings, HVAC ducts)

Most expensive

CMR / Riser

Riser-Rated

  • Fire-resistant
  • Prevents fire spread between floors
  • Moderate smoke
  • Required in: Vertical runs between floors

Mid-range cost

CM / PVC

General Purpose

  • Standard PVC jacket
  • Not fire-rated
  • Can produce toxic smoke
  • Use in: Patch cables, non-plenum horizontal

Least expensive

‚ö†Ô∏è Code Violations

Installing PVC cable in plenum spaces is a fire code violation and can result in failed inspections, fines, or worse - a real fire hazard!

üè¢ Where is the Plenum?

Look up at the drop ceiling in an office. That space between the tiles and the real ceiling? That's often used for HVAC air return - that's a plenum space. Air circulates there, so fire/smoke would spread quickly through the building.

Common Installation Mistakes

Learn from others' errors!

‚ùå DON'T Do This

  • Exceed bend radius - Damages internal conductors
  • Untwist pairs too much - Max 0.5" for Cat6
  • Run parallel to power cables - EMI interference
  • Over-tighten cable ties - Crushes cables
  • Pull cables too hard - Stretches conductors
  • Exceed 100m total - Including patch cables!

The 90/10 Rule

For a 100m Ethernet channel:

  • 90m maximum - Permanent horizontal cable
  • 10m total - Patch cables (5m each end)

‚úÖ DO This Instead

  • Maintain bend radius - 4x cable diameter minimum
  • Keep pairs twisted - Close to termination point
  • Separate from power - Or cross at 90¬∞ angles
  • Use velcro straps - Adjustable, won't crush
  • Use pull strings - Protects the cable
  • Label everything - Both ends of every cable!

EMI Sources to Avoid

  • Fluorescent light ballasts
  • Electric motors
  • Power cables (especially high voltage)
  • Elevators
  • Generators

Cable Testing & Certification

What the tests actually tell you

Copper Cable Tests

  • Wire Map - Correct pin-to-pin connections
  • Continuity - Complete circuit, no breaks
  • Length - Within 100m limit
  • Attenuation - Signal loss over distance
  • NEXT - Near-End Crosstalk
  • FEXT - Far-End Crosstalk
  • Return Loss - Signal reflection

Common Failures

  • Open - Broken conductor
  • Short - Wires touching
  • Split Pair - Wrong wires paired
  • Reversed Pair - + and - swapped

Fiber Optic Tests

  • Insertion Loss - Power lost in dB
  • Continuity - Light passes through
  • OTDR Trace - Distance to faults
  • ORL - Optical Return Loss
  • Visual Inspection - Clean connectors!

Clean Fiber = Happy Fiber

80% of fiber problems are dirty connectors. Always clean fiber ends before connecting!

  • Use lint-free wipes
  • Isopropyl alcohol (99%)
  • Inspect with fiber scope
