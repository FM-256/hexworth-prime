DNS Header Reference

Interactive DNS packet header structure

The Domain Name System (DNS) is a hierarchical naming system that translates human-readable domain names to IP addresses. DNS messages contain a header section followed by question, answer, authority, and additional sections. Click any field below to learn more.

DNS Header Structure (12 bytes)

Identification 16 bits QR 1 Opcode 4 AA 1 TC 1 RD 1 RA 1 Z 3 RCode 4
QDCount 16 bits ANCount 16 bits
NSCount 16 bits ARCount 16 bits
