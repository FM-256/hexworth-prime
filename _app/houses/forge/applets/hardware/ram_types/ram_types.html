üíæ What is RAM?

Random Access Memory (RAM) is volatile memory that temporarily stores data the CPU needs quick access to. Unlike storage drives, RAM loses all data when power is removed.

üí° A+ Exam Tip: RAM is measured in GB (gigabytes) and speed in MHz or MT/s (megatransfers per second). Know the physical differences between DIMM and SO-DIMM modules!

üõ°Ô∏è ECC vs Non-ECC Memory

Non-ECC (Standard)

  • Used in consumer desktops & laptops
  • No error detection/correction
  • Lower cost
  • Faster performance

ECC (Error-Correcting Code)

  • Used in servers & workstations
  • Detects and corrects single-bit errors
  • Higher reliability for critical systems
  • Requires compatible motherboard/CPU
