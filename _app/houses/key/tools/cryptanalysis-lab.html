üìä Frequency Analysis

Analyze letter frequency in ciphertext to break monoalphabetic substitution ciphers. This technique exploits the non-uniform distribution of letters in natural language.

How it works:
In English, 'E' appears ~12.7% of the time, 'T' ~9.1%, 'A' ~8.2%, etc. By comparing ciphertext frequencies to expected frequencies, we can deduce the substitution mapping.

üéÇ Birthday Attack Probability

Calculate the probability of hash collisions based on the birthday paradox. For an n-bit hash, collision probability reaches 50% after approximately 2^(n/2) operations.

üîÆ Padding Oracle Attack Visualizer

Step-by-step visualization of how padding oracle attacks decrypt AES-CBC ciphertext by exploiting servers that reveal padding validity. This attack was used in POODLE to break SSLv3.

Attack Overview:
1. Modify ciphertext bytes systematically
2. Submit to server (the "oracle")
3. Server reveals if padding is valid
4. Use this information to deduce plaintext byte-by-byte

üîÑ CBC Bit-Flipping Attack

Demonstrates CBC mode malleability: flipping a bit in ciphertext block Ci causes a predictable bit flip in plaintext block Pi+1. This allows attackers to modify encrypted data without knowing the key.

In CBC mode: Pi+1 = Decrypt(Ci+1) ‚äï Ci
Modifying Ci directly affects Pi+1 through XOR.
