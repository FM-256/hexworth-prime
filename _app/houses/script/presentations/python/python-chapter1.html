Understanding Data Types

Data Type: A classification that determines the kind of data a variable can hold and what operations can be performed on it.

üìã Purpose

Data types help the computer understand and manage different kinds of information.

‚ö†Ô∏è Importance

Using the correct data type ensures operations are performed correctly, preventing errors and ensuring data integrity.

üî§ Primitive vs Non-primitive

Primitive types (int, float) are basic. Non-primitive types (list, dict) are built from primitives.

Python Data Types

üî¢ int

What is it? Represents whole numbers (positive, negative, or zero).

Use Case: Count things like age, quantity, or score.

age = 25
items = 4
print(age * items) # 100
üìê float

What is it? Represents decimal numbers for precise values.

Use Case: Prices, measurements, scientific calculations.

price = 19.99
quantity = 3
total = price * quantity
üìù str

What is it? A string of characters enclosed in quotes.

Use Case: Names, messages, input, all text-related data.

name = "Alice"
print("Hello, " + name)
‚úÖ bool

What is it? Holds one of two values: True or False.

Use Case: Logic for conditionals like 'is active' or 'is valid'.

is_logged_in = True
if is_logged_in:
    print("Welcome back!")
üìã list

What is it? An ordered, changeable collection of items.

Use Case: To-do lists, shopping carts, results.

fruits = ["apple", "banana"]
fruits.append("cherry")
print(fruits)
üì¶ tuple

What is it? An ordered, unchangeable collection.

Use Case: Fixed data like coordinates or configuration pairs.

location = (40.7128, -74.0060)
print(location[0])
üóÇÔ∏è dict

What is it? Stores data as key-value pairs.

Use Case: User profiles, configurations, labeled data.

user = {"name": "Alice", "age": 30}
print(user["name"])
üîµ set

What is it? Unordered collection of unique elements.

Use Case: Remove duplicates or quickly check membership.

emails = {"a@example.com", "b@example.com"}
print(len(emails))

Advanced Data Types

üî£ complex

What is it? Numbers with real and imaginary parts.

Use Case: Electrical engineering, physics, complex math.

z = 3 + 4j
print(z.real) # 3.0
print(z.imag) # 4.0
üîÑ range

What is it? Represents a sequence of numbers, often in loops.

Use Case: Generating indices or repeating code.

for i in range(5):
    print(i)
üíæ bytes

What is it? Immutable sequence of raw binary data.

Use Case: File storage, network transmission, media data.

data = bytes(5)
print(data) # b'\x00\x00\x00\x00\x00'
‚ùÑÔ∏è frozenset

What is it? Immutable version of a set.

Use Case: Constant, unchangeable set of unique items.

constants = frozenset([3.14, 2.718])
print(constants)
