What Is Flow Control?

Flow control is how your program makes decisions and repeats actions based on conditions.

Logical Operators

and

Both conditions must be True.

age >= 18 and has_id # True only if BOTH true
or

At least one condition must be True.

is_admin or is_owner # True if EITHER true
not

Inverts the boolean value.

not is_blocked # True becomes False

Conditionals: if, elif, else

Making Decisions
# Basic if statement age = 20 if age >= 18: print("You can vote!") # if-else if age >= 21: print("Can drink in US") else: print("Not yet!") # if-elif-else chain score = 85 if score >= 90: grade = "A" elif score >= 80: grade = "B" elif score >= 70: grade = "C" else: grade = "F" print(f"Grade: {grade}") # Grade: B

Loops: for and while

for loop

Iterate over a sequence (list, range, string).

for i in range(5): print(i) # 0,1,2,3,4 fruits = ["apple", "banana"] for fruit in fruits: print(fruit)
while loop

Repeat while a condition is True.

count = 0 while count < 5: print(count) count += 1 # Careful: infinite loops!

Loop Control: break, continue, pass

break

Exit the loop immediately.

for i in range(10): if i == 5: break print(i) # 0,1,2,3,4
continue

Skip to the next iteration.

for i in range(5): if i == 2: continue print(i) # 0,1,3,4
pass

Do nothing (placeholder).

if condition: pass # TODO: add code
