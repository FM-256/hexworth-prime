Chapter 2: Strings & Text Handling

What is a String?

A string is a sequence of characters enclosed in quotes. It can include letters, numbers, spaces, and symbols.

Basic String Example
text = "Hello, world!" print(text) # Hello, world!
'Single Quotes'

Strings can use single quotes.

msg = 'Hello'
"Double Quotes"

Or double quotes - both work the same.

msg = "Welcome!"
'''Triple Quotes'''

For multi-line strings and docstrings.

bio = '''Line 1 Line 2'''

String Types

r"Raw String"

What: Treats backslashes as literal characters.

Use: File paths, regex patterns.

path = r"C:\Users\Name"
b"Byte String"

What: Creates a byte object.

Use: Binary data, network streams.

data = b"hello"
f"F-String"

What: Formatted string with embedded expressions.

Use: Dynamic messages, variable embedding.

f"Hello, {name}!"

F-Strings: Modern String Formatting

F-strings (formatted string literals) are the modern, readable way to embed variables and expressions in strings.

F-String Examples
# Basic variable embedding name = "Alice" print(f"Hello, {name}!") # Hello, Alice! # Expressions inside f-strings a = 5 b = 3 print(f"Sum is {a + b}") # Sum is 8 # Formatting numbers total = 29.99 print(f"Your total is ${total:.2f}") # Your total is $29.99 # Old way vs new way print("Hello, {}".format(name)) # Old .format() method print(f"Hello, {name}") # New f-string (cleaner!)
