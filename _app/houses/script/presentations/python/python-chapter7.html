Chapter 7: File Handling

Opening Files

Python's built-in open() function allows you to read from and write to files.

ModeDescriptionCreates File?
'r'Read (default) - file must existNo
'w'Write - overwrites fileYes
'a'Append - adds to endYes
'x'Create - fails if existsYes
'r+'Read and WriteNo
'b'Binary mode (add to others)-

The "with" Statement

‚úÖ Best Practice

Always use with when working with files. It automatically closes the file, even if an error occurs!

Reading Files
# Best way: with statement with open("data.txt", "r") as file: content = file.read() print(content) # File automatically closed here! # Read all lines into a list with open("data.txt") as file: lines = file.readlines() for line in lines: print(line.strip()) # Memory-efficient: iterate directly with open("data.txt") as file: for line in file: print(line.strip())

Writing to Files

Writing Methods
# Write (overwrites!) with open("output.txt", "w") as file: file.write("Hello, World!\n") file.write("Second line\n") # Write multiple lines lines = ["Line 1\n", "Line 2\n", "Line 3\n"] with open("output.txt", "w") as file: file.writelines(lines) # Append (doesn't overwrite) with open("log.txt", "a") as file: file.write("New log entry\n")

‚ö†Ô∏è Warning

Mode 'w' will overwrite the file if it exists! Use 'a' to append instead.

Working with CSV Files

CSV Module
import csv # Reading CSV with open("users.csv") as file: reader = csv.reader(file) for row in reader: print(row) # ['name', 'email', ...] # Reading as dictionary with open("users.csv") as file: reader = csv.DictReader(file) for row in reader: print(row["name"], row["email"]) # Writing CSV with open("output.csv", "w", newline="") as file: writer = csv.writer(file) writer.writerow(["name", "age"]) writer.writerow(["Alice", 30])
