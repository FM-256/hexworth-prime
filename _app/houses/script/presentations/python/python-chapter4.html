Chapter 4: Functions

What is a Function?

A function is a reusable block of code that performs a specific task. Functions help organize code, reduce repetition, and make programs easier to maintain.

Basic Function Structure
def greet(name): """Greet a user by name.""" return f"Hello, {name}!" # Call the function message = greet("Alice") print(message) # Hello, Alice!

Function Parameters

Positional Arguments

Arguments matched by position.

def add(a, b): return a + b add(5, 3) # a=5, b=3
Keyword Arguments

Arguments matched by name.

def greet(name, msg): print(f"{msg}, {name}") greet(msg="Hi", name="Bob")
Default Values

Parameters with default values.

def greet(name, msg="Hello"): print(f"{msg}, {name}") greet("Alice") # Hello, Alice
*args & **kwargs

Variable number of arguments.

def sum_all(*args): return sum(args) sum_all(1,2,3,4) # 10

Return Values

Returning Data
# Single return value def square(n): return n ** 2 # Multiple return values (tuple) def divide(a, b): quotient = a // b remainder = a % b return quotient, remainder q, r = divide(17, 5) # q=3, r=2 # No return = returns None def say_hi(): print("Hi!") # implicitly returns None

Lambda Functions

Anonymous, one-line functions for simple operations.

Lambda Syntax
# Regular function def double(x): return x * 2 # Same as lambda double = lambda x: x * 2 # Common with map, filter, sorted numbers = [1, 2, 3, 4] doubled = list(map(lambda x: x * 2, numbers)) # [2, 4, 6, 8]

Scope: Local vs Global

üìå Key Concept

Variables inside a function are local - they only exist within that function. Variables outside are global.

x = 10 # Global def example(): y = 5 # Local print(x) # Can read global print(y) # Local variable example() print(x) # 10 - works # print(y) # Error! y is local
