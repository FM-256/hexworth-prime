Chapter 6: Dictionaries

What is a Dictionary?

A dictionary stores data as key-value pairs. Think of it like a real dictionary: the word is the key, the definition is the value.

Dictionary Basics
# Creating dictionaries user = { "name": "Alice", "age": 30, "email": "alice@example.com" } # Accessing values print(user["name"]) # Alice print(user.get("age")) # 30 # get() with default (avoids KeyError) print(user.get("phone", "N/A")) # N/A

Dictionary Operations

Modifying Dictionaries
# Adding/updating values user["phone"] = "555-1234" # Add new user["age"] = 31 # Update existing # Removing items del user["email"] # Remove key phone = user.pop("phone") # Remove & return # Check if key exists if "name" in user: print("Name found!") # Get all keys/values print(user.keys()) # dict_keys(['name', 'age']) print(user.values()) # dict_values(['Alice', 31]) print(user.items()) # dict_items([('name','Alice'),...])

Dictionary Methods

.get(key, default)

Safely get value with fallback.

user.get("age", 0)
.update(dict)

Merge another dict.

user.update({"city": "NYC"})
.pop(key)

Remove & return value.

val = user.pop("age")
.clear()

Remove all items.

user.clear() # {}

Iterating Over Dictionaries

Looping Through Dicts
user = {"name": "Alice", "age": 30} # Loop through keys for key in user: print(key) # name, age # Loop through values for value in user.values(): print(value) # Alice, 30 # Loop through key-value pairs for key, value in user.items(): print(f"{key}: {value}") # name: Alice # age: 30

Nested Dictionaries

Dicts Inside Dicts
users = { "user1": { "name": "Alice", "email": "alice@example.com" }, "user2": { "name": "Bob", "email": "bob@example.com" } } # Accessing nested values print(users["user1"]["name"]) # Alice

Dictionary Comprehensions

# Create dict from sequence squares = {x: x**2 for x in range(5)} # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16} # Filter and transform prices = {"apple": 1.0, "banana": 0.5, "cherry": 2.0} expensive = {k: v for k, v in prices.items() if v > 0.75} # {'apple': 1.0, 'cherry': 2.0}
