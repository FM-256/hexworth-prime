Lists: The Workhorse Collection

Lists are ordered, mutable sequences. Use them when you need to add, remove, or change items.

List Basics
# Creating lists fruits = ["apple", "banana", "cherry"] numbers = [1, 2, 3, 4, 5] mixed = [1, "hello", 3.14, True] # Accessing items print(fruits[0]) # apple (first) print(fruits[-1]) # cherry (last) # Slicing print(fruits[0:2]) # ['apple', 'banana']

List Methods

.append(item)

Add item to end.

fruits.append("orange")
.insert(i, item)

Insert at position.

fruits.insert(1, "mango")
.remove(item)

Remove first occurrence.

fruits.remove("banana")
.pop(i)

Remove & return item at index.

last = fruits.pop() # removes last
.sort()

Sort list in place.

numbers.sort() numbers.sort(reverse=True)
.reverse()

Reverse order in place.

fruits.reverse()

List Comprehensions

A concise way to create lists from existing sequences.

Comprehension Syntax
# Traditional loop squares = [] for x in range(5): squares.append(x ** 2) # List comprehension (same result!) squares = [x ** 2 for x in range(5)] # [0, 1, 4, 9, 16] # With condition evens = [x for x in range(10) if x % 2 == 0] # [0, 2, 4, 6, 8]

Tuples: Immutable Lists

Tuples are like lists but cannot be modified after creation. Use them for fixed data.

Tuple Basics
# Creating tuples point = (10, 20) colors = ("red", "green", "blue") single = (42,) # Note the comma! # Accessing (same as lists) print(point[0]) # 10 # Tuple unpacking x, y = point print(f"x={x}, y={y}") # x=10, y=20 # Can't modify! # point[0] = 5 # TypeError!
