SCRIPTING BRIEFING

Why Shell Scripting?

Automation is power. Shell scripts let you:

  • Automate repetitive tasks
  • Chain commands together
  • Create reusable tools
  • Schedule operations

Script Anatomy

#!/bin/bash
# This is a comment
NAME="Operator"
echo "Hello, $NAME"

Running Scripts

$ ./script.sh
Execute directly (needs +x)
$ bash script.sh
Run via bash interpreter

Variables & Echo

$ echo $HOME
/home/operator
$ echo "Current user: $(whoami)"
Current user: operator
SCRIPT NOTE:

Always use #!/bin/bash as the first line (shebang).

PRO TIP:

Use bash -x script.sh to debug scripts (shows each command as it runs).
