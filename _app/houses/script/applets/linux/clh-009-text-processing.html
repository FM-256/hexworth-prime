ANALYST TOOLKIT

The Mission

You've intercepted server logs from a compromised host. Extract and analyze the data to identify suspicious activity patterns.

Text processing is the backbone of log analysis and forensics.

cut - Extract Columns

$ cut -d ' ' -f 1 file.txt
-d = delimiter, -f = field number
$ cut -d ':' -f 1,3 /etc/passwd
Extract fields 1 and 3 (colon-delimited)

sort - Order Data

$ sort file.txt
Alphabetical sort
$ sort -n file.txt
Numeric sort

uniq - Find Unique/Duplicates

$ sort file.txt | uniq
Remove duplicates (must sort first!)
$ sort file.txt | uniq -c
Count occurrences

awk - Pattern Processing

$ awk '{print $1}' file.txt
Print first column (whitespace-delimited)
$ awk -F: '{print $1}' /etc/passwd
-F sets delimiter

sed - Stream Editor

$ sed 's/old/new/g' file.txt
Replace all 'old' with 'new'
$ sed 's/[0-9]/X/g' file.txt
Replace all digits with X
PIPELINE POWER:

Chain commands with | for complex analysis:
cut -d' ' -f1 log | sort | uniq -c | sort -rn

REMEMBER:

uniq only removes ADJACENT duplicates. Always sort first!
