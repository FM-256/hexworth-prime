Amazon RDS
Relational Database
Amazon Relational Database Service (RDS) makes it easy to set up, operate, and scale a relational database in the cloud. It provides cost-efficient and resizable capacity while automating time-consuming administration tasks.
Supported Engines
  • Amazon Aurora (MySQL/PostgreSQL compatible)
  • MySQL
  • PostgreSQL
  • MariaDB
  • Oracle Database
  • Microsoft SQL Server
Key Features
  • Automated backups and snapshots
  • Multi-AZ deployments for high availability
  • Read replicas for scalability
  • Automatic software patching
  • Point-in-time recovery
  • Encryption at rest and in transit
Best For
  • Traditional applications with complex queries
  • Applications requiring ACID compliance
  • Existing applications migrating to cloud
  • Enterprise applications

Amazon DynamoDB
NoSQL Database
Amazon DynamoDB is a fully managed NoSQL database service that provides fast and predictable performance with seamless scalability. It supports both document and key-value data models.
Key Characteristics
  • Single-digit millisecond latency
  • Fully managed with automatic scaling
  • Built-in security and backup
  • Global tables for multi-region replication
  • In-memory caching with DAX
Key Features
  • On-demand and provisioned capacity modes
  • Point-in-time recovery
  • Streams for change data capture
  • Fine-grained access control
  • Encryption at rest by default
  • TTL for automatic data expiration
Best For
  • High-traffic web applications
  • Gaming applications
  • IoT applications
  • Mobile backends
  • Real-time bidding platforms

Amazon Redshift
Data Warehouse
Amazon Redshift is a fast, fully managed data warehouse that makes it simple and cost-effective to analyze all your data using standard SQL and existing Business Intelligence tools.
Key Characteristics
  • Columnar storage for fast queries
  • Massively parallel processing (MPP)
  • Petabyte-scale data warehouse
  • Standard SQL interface
  • Integration with BI tools
Key Features
  • Redshift Spectrum for S3 queries
  • Concurrency scaling
  • Automatic backups and snapshots
  • Result caching for faster queries
  • Machine learning insights
  • Federated queries
Best For
  • Business intelligence and analytics
  • Big data processing
  • Complex analytical queries
  • Historical data analysis
  • Data consolidation from multiple sources

Amazon ElastiCache
In-Memory Cache
Amazon ElastiCache is a fully managed in-memory data store and cache service supporting Redis and Memcached. It improves application performance by allowing retrieval of information from fast, managed, in-memory caches.
Supported Engines
  • Redis - Advanced data structures and persistence
  • Memcached - Simple, high-performance caching
Key Features
  • Sub-millisecond latency
  • Automatic failover and recovery
  • In-memory data replication
  • Backup and restore (Redis)
  • Multi-AZ with automatic failover
  • Read replicas for scaling reads
Best For
  • Session storage
  • Database query result caching
  • Leaderboards and counting
  • Real-time analytics
  • Message queues and pub/sub

Amazon Aurora
Cloud-Native Relational
Amazon Aurora is a MySQL and PostgreSQL-compatible relational database built for the cloud. It combines the performance and availability of high-end commercial databases with the simplicity and cost-effectiveness of open source.
Key Characteristics
  • 5x faster than MySQL, 3x faster than PostgreSQL
  • 99.99% availability
  • Storage auto-scaling up to 128TB
  • Up to 15 read replicas
  • Global database spanning multiple regions
Key Features
  • Aurora Serverless for variable workloads
  • Continuous backup to S3
  • Point-in-time recovery
  • Fast cloning
  • Advanced security features
  • Performance Insights
Best For
  • Enterprise applications requiring high availability
  • SaaS applications
  • Applications needing global reach
  • Variable or unpredictable workloads (Serverless)

Amazon DocumentDB
Document Database
Amazon DocumentDB is a fast, scalable, highly available, and fully managed document database service that supports MongoDB workloads. It's designed to give you the performance, scalability, and availability you need.
Key Characteristics
  • MongoDB-compatible
  • Fully managed service
  • Scalable storage up to 64TB
  • Up to 15 read replicas
Key Features
  • Automated backups and snapshots
  • Multi-AZ deployment
  • Encryption at rest and in transit
  • Point-in-time recovery
  • Integration with AWS services
Best For
  • Content management systems
  • Catalog management
  • User profiles
  • Mobile and web applications

Real-World Use Cases

E-Commerce Platform
RDS + ElastiCache
Use RDS for product catalog, orders, and customer data. ElastiCache for session management and frequently accessed product information to reduce database load.
Mobile Gaming Backend
DynamoDB + ElastiCache
DynamoDB for player profiles, game state, and leaderboards with automatic scaling. ElastiCache for real-time leaderboard updates and session data.
Business Intelligence Platform
Redshift + S3
Redshift as the central data warehouse for analyzing petabytes of data from multiple sources. Use Redshift Spectrum to query data directly in S3.
Content Management System
DocumentDB + S3
DocumentDB for storing flexible content schemas, metadata, and user-generated content. S3 for media files with CloudFront for delivery.
SaaS Application
Aurora + DynamoDB
Aurora for relational data with high availability and global reach. DynamoDB for user sessions, preferences, and high-velocity event data.
IoT Data Pipeline
DynamoDB + Redshift
DynamoDB for ingesting high-velocity IoT sensor data. Archive to S3 and analyze in Redshift for long-term trends and insights.
Financial Trading Platform
ElastiCache + RDS
ElastiCache for real-time market data and ultra-low latency trading operations. RDS for transaction history and compliance records.
Social Media Analytics
DynamoDB + Redshift
DynamoDB for storing user posts, comments, and interactions. Stream data to Redshift for trend analysis and user behavior insights.
Healthcare Records System
Aurora + S3
Aurora for patient records, appointments, and billing with encryption and compliance. S3 for medical imaging with lifecycle policies.
