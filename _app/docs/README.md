## üöÄ Getting Started

### **Option 1: Local Use (Recommended)**
1. Extract the zip file to a folder
2. Open `catalog.html` in any modern web browser
3. Navigate to presentations, labs, or simulators
4. All content works offline - no internet required!

### **Option 2: Web Server Deployment**
1. Upload all files to a web server
2. Maintain the folder structure
3. Access via `https://your-domain.com/catalog.html`
4. Students can access remotely

### **Option 3: LMS Integration**
1. Upload as SCORM package (files are standalone HTML)
2. Link to individual presentations from LMS
3. Provide lab handout PDF for assignments
